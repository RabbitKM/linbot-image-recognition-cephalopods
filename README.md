# LINE Bot Designer & Image Recognition (Cephalopods)
> TibaMe AI班團體專題

分別串接LINE Bot、AI模型與GCP，透過團隊訓練好的模型辨識易混淆的常見五種頭足類，並蒐集用戶資料做視覺化分析。

※本人作為組長負責：照片拍攝、爬蟲、資料清洗、LINE Bot程式撰寫部屬、用戶資料視覺化

## ◆專案目的
生活中常見的頭足類長得相似容易混淆，因俗名很多也常搞不清楚如何稱呼。希望開發一款快速準確辨識的行動應用工具，利用AI協助使用者快速分辨台灣市場常見的頭足類名稱。
